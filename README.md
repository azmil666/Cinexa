# 🎬 Cinexa

**Discover movies, beautifully.**

Cinexa is a modern movie discovery web app that allows users to search for films, explore details, and enjoy a clean, minimal interface inspired by premium products like Vercel and Linear.

---

## 🚀 Live Demo

👉 https://cinexa-sepia.vercel.app

---

## ✨ Features

* 🔍 **Instant Movie Search**
  Search movies in real-time using the OMDb API

* 🎥 **Detailed Movie Info**
  View ratings, plot, year, and poster in a clean popup modal

* ⚡ **Fast & Responsive**
  Smooth experience across devices

* 🖼️ **Smart Image Handling**
  Handles missing posters gracefully

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **API:** OMDb API
* **HTTP Client:** Axios
* **Deployment:** Vercel

## ⚙️ Setup & Installation

1. **Clone the repo**

```bash
git clone https://github.com/azmil666/Cinexa.git
cd Cinexa
```

2. **Install dependencies**

```bash
npm install
```

3. **Add environment variables**

Create a `.env` file in root:

```env
REACT_APP_OMDB_API_KEY=your_api_key_here
```

4. **Run the app**

```bash
npm start
```
